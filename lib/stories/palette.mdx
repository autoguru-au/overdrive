# Palette

The AutoGuru Design System 2026 colour ramps. Values are sourced from the
[Figma palette](https://www.figma.com/design/ZkQlQcJkF7NTnZomVrPRN5/AutoGuru-Design-System-2026?node-id=360-12804)
bound variables — `lib/themes/base/colours.ts` is the single source of truth in
code.

export const ramp = (hue) =>
  Object.entries(colourMapWithoutWhite[hue])
    .reverse()
    // the trailing space in the key stops JS from re-sorting integer-like
    // keys ascending, keeping the 900 → 100 display order
    .reduce(
      (shades, [step, hex]) => ({ ...shades, [`${step} `]: hex.toUpperCase() }),
      {},
    );

<ColorPalette>
  <ColorItem title="Green" colors={ramp('green')} />
  <ColorItem title="Gray" colors={ramp('gray')} />
  <ColorItem title="Yellow" colors={ramp('yellow')} />
  <ColorItem title="Blue" colors={ramp('blue')} />
  <ColorItem title="Red" colors={ramp('red')} />
  <ColorItem title="White" colors={{ white: colourMap.white.toUpperCase() }} />
</ColorPalette>

> **Deprecated:** the standalone `black` ramp was removed in Design System 2026
> — "Tarmac Black" is now `gray/900`. The `black*` tokens (and
> `--od-color-gamut-black-*` CSS vars) are aliased to the `gray` ramp for
> backwards compatibility and will be removed in a future major release. Use
> `gray*` instead.
