# Contrast Guide

The design-approved text-on-background combinations for the Design System 2026
[Colour Palette](?path=/docs/foundation-colour-palette--docs). Every `text`
combination passes WCAG AA for normal text (4.5:1); `icon only` and
`inactive only` are conditional exceptions.

**Do not use any colour combination that is not listed here.**

## How the ramps work

- **Gray, blue and red** — white text works on steps **500–900**. Reversed, the
  same dark steps work as text on white (blue down to 600, red down to 700).
- **Green** — white text only on **800–900**; use `gray-900` text on 300–600.
- **Yellow** — **never white text**. Use `gray-900` text on any step, or
  `red-900` on 100–500.
- **Steps 100–200** — tinted backgrounds; pair with the same hue's 900 text (or
  `gray-900`).
- **Conditional** — `gray-400` on white is for inactive elements only;
  `green-600` and `red-600` on white are for icons only, never text.

## Approved combinations

One grid per ramp: rows are foreground (text) colours, columns are backgrounds.
Coloured cells are approved — each shows the live pair, its contrast ratio, and
the WCAG level. Grey `✕` cells are not approved, even where the raw ratio
passes.

export const steps = Object.keys(colourMapWithoutWhite.gray);

export const ratioOf = (foreground, background) =>
  1 /
  getContrastRatio(
    contrastGuideColour(foreground),
    contrastGuideColour(background),
  );

export const tokenOrder = (token) => {
  if (token === 'white') return 0;
  if (token === 'gray-900') return 1;
  const [, step] = token.split('-');
  return 2 + (900 - Number(step));
};

export const foregroundsFor = (hue) =>
  [...new Set(contrastGuide[hue].map((combo) => combo.foreground))].sort(
    (a, b) => tokenOrder(a) - tokenOrder(b),
  );

export const usageFor = (hue) =>
  new Map(
    contrastGuide[hue].map((combo) => [
      `${combo.foreground}|${combo.background}`,
      combo.usage,
    ]),
  );

export const Cell = ({ foreground, background, usage }) => {
  if (!usage) {
    return (
      <div
        style={{
          background: '#fafbfc',
          borderRadius: 6,
          minHeight: 56,
          display: 'flex',
          alignItems: 'center',
          justifyContent: 'center',
          color: '#d4d9dd',
          fontSize: 12,
        }}
      >
        ✕
      </div>
    );
  }
  const value = ratioOf(foreground, background);
  const badge =
    usage === 'icon-only'
      ? 'icon only'
      : usage === 'inactive-only'
        ? 'inactive'
        : value >= 7
          ? 'AAA'
          : 'AA';
  return (
    <div
      style={{
        background: contrastGuideColour(background),
        color: contrastGuideColour(foreground),
        borderRadius: 6,
        boxShadow: 'inset 0 0 0 1px rgba(33, 35, 56, 0.12)',
        minHeight: 56,
        padding: '6px 8px',
        display: 'flex',
        flexDirection: 'column',
        justifyContent: 'center',
        textAlign: 'center',
      }}
    >
      <div style={{ fontWeight: 700, fontSize: 14, lineHeight: 1.3 }}>Aa</div>
      <div style={{ fontSize: 10, lineHeight: 1.4, opacity: 0.9 }}>
        {value.toFixed(2)} · {badge}
      </div>
    </div>
  );
};

export const headStyle = {
  fontSize: 11,
  color: '#5c6172',
  alignSelf: 'end',
  textAlign: 'center',
  paddingBottom: 4,
};

export const Grid = ({ hue }) => {
  const backgrounds = ['white', ...steps.map((step) => `${hue}-${step}`)];
  const foregrounds = foregroundsFor(hue);
  const usage = usageFor(hue);
  return (
    <div style={{ margin: '24px 0 40px' }}>
      <div
        style={{
          fontSize: 16,
          fontWeight: 700,
          color: '#212338',
          textTransform: 'capitalize',
          marginBottom: 8,
        }}
      >
        {hue}
      </div>
      <div style={{ overflowX: 'auto' }}>
        <div
          style={{
            display: 'grid',
            gridTemplateColumns: `88px repeat(${backgrounds.length}, minmax(60px, 1fr))`,
            gap: 4,
            minWidth: 720,
          }}
        >
          <div style={{ ...headStyle, textAlign: 'left' }}>text ↓ on →</div>
          {backgrounds.map((background) => (
            <div key={background} style={headStyle}>
              {background === 'white' ? 'white' : background.split('-')[1]}
            </div>
          ))}
          {foregrounds.map((foreground) => (
            <Fragment key={foreground}>
              <div
                style={{
                  fontSize: 11,
                  color: '#212338',
                  fontWeight: 600,
                  alignSelf: 'center',
                }}
              >
                {foreground}
              </div>
              {backgrounds.map((background) => (
                <Cell
                  key={background}
                  foreground={foreground}
                  background={background}
                  usage={usage.get(`${foreground}|${background}`)}
                />
              ))}
            </Fragment>
          ))}
        </div>
      </div>
    </div>
  );
};

{Object.keys(contrastGuide).map((hue) => (

<Grid key={hue} hue={hue} />
))}
